# ASP.NET Core MVC - TDD

Projeto destinado aos desenvolvedores ASP.NET Core MVC, com o objetivo de apresentar os recursos para desenvolver códigos a partir da implementação de testes. O TDD é uma técnica utilizada para guiar o desenvolvimento com base nos testes, então você cria o teste para uma função que ainda não foi desenvolvida e apenas após o teste, você desenvolve seu código propriamente dito. Ao final do seu desenvolvimento, você terá um sistema com 100% (ou bem próximo disto) de cobertura de testes.

Esta abordagem concede ao desenvolvedor maior acertividade no desenvolvimento de sistemas e facilita a manutenção do código, já que você se preocupou em refatorar seu código para deixá-lo simples e claro, seguindo as premissas do TDD. Além disso, você evita muito retrabalho, devido ao fato de você ter feito seus testes antes mesmo de desenvolver seu código.
